Instrucciones de uso:
  Importar el proyecto desde eclipse: File>Import>General>Existing projects into workspace 
  Seleccionar la carpeta donde se encuentre descomprimido el proyecto
  
  
  Para ejecutar el servidor:
    Run as>Run on Server
    Seleccionar un runtime con un servidor Tomcat
    
  Para ejecutar el cliente:
  	Run as>Java Application
  	Seleccionar el la clase Cliente de es.dss.interfaz como clase main
	

Entorno de trabajo y dependencias;
- JDK 1.8.0_111
- Eclipse Java EE IDE for Web Developers. Version: Mars.2 Release (4.5.2)
- Apache Tomcat 8.0.14
- Debian 8.6
- EclipseLink 2.6.4
- Derby 10.13.1.1
- JPA 2.1
